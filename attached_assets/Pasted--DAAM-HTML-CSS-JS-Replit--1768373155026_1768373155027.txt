ابنِ موقع ويب جديد باسم DAAM داخل مشروع HTML/CSS/JS على Replit.

المطلوب النهائي:
- نظام مستقل بسيط يعمل فورًا عبر زر Run.
- 3 صفحات: تسجيل دخول للطلاب فقط + ساحة منشورات + صفحة معلم AI (نسخة تجريبية).
- واجهة ثنائية اللغة (عربي/English) بزر تبديل لغة.
- تحقق مبدئي: يسمح فقط بإيميل جامعي ينتهي بـ @utas.edu.om (اجعل الدومين في متغير يسهل تغييره لاحقًا).
- تخزين مؤقت محلي عبر localStorage (لا قاعدة بيانات الآن).

1) أنشئ/حدّث الملفات التالية:
- index.html (يعيد التوجيه تلقائيًا إلى login.html)
- login.html
- feed.html
- tutor.html
- style.css
- app.js

2) المتطلبات التفصيلية:

A) index.html
- لا يحتوي إلا على إعادة توجيه تلقائية:
  - إذا كان localStorage يحتوي daam_user انتقل إلى feed.html
  - غير ذلك انتقل إلى login.html

B) login.html
- عنوان: DAAM
- وصف قصير: منصة تواصل للطلاب + معلم AI
- حقل Email وزر "دخول / Login"
- رسالة خطأ واضحة بالعربي والإنجليزي حسب اللغة المختارة
- شروط الدخول:
  - الإيميل يجب أن يحتوي @
  - يجب أن ينتهي بالدومين المسموح (متغير allowedDomain = "utas.edu.om")
- عند نجاح الدخول:
  - خزّن الإيميل في localStorage بالمفتاح daam_user
  - انتقل إلى feed.html
- زر تبديل اللغة: AR / EN
  - خزّن اللغة في localStorage بالمفتاح daam_lang (ar أو en)

C) feed.html (ساحة DAAM)
- قبل عرض الصفحة:
  - إذا لا يوجد daam_user في localStorage => انتقل إلى login.html
- واجهة:
  - شريط علوي يعرض: البريد + زر "المعلم AI" + زر "خروج"
  - قسم كتابة منشور: textarea + زر "نشر / Post"
  - قائمة منشورات تظهر الأحدث أولًا
- التخزين:
  - خزّن المنشورات في localStorage بالمفتاح daam_posts_v1 بصيغة JSON Array
  - كل منشور يحتوي:
    - id (random)
    - authorEmail
    - content
    - createdAt (ISO string)
- زر "خروج / Logout":
  - احذف daam_user
  - انتقل إلى login.html

D) tutor.html (معلم DAAM - نسخة تجريبية)
- قبل عرض الصفحة:
  - إذا لا يوجد daam_user => انتقل إلى login.html
- واجهة محادثة بسيطة:
  - رسالة افتتاحية من DAAM
  - حقل سؤال + زر إرسال
  - عند إرسال سؤال:
    - أضف السؤال للمحادثة
    - أضف رد تجريبي ثابت:
      - AR: "تم استلام سؤالك. سيتم تفعيل الذكاء الاصطناعي وربطه بمواد DAAM قريبًا."
      - EN: "Your question is received. AI tutor will be enabled and connected to DAAM materials soon."
- زر "رجوع للساحة / Back to Feed" ينقل إلى feed.html

E) style.css
- تصميم داكن نظيف:
  - خلفية داكنة
  - بطاقة/Container في الوسط
  - أزرار واضحة
  - أقسام مرتبة
- دعم RTL للعربية و LTR للإنجليزية عبر تغيير اتجاه الصفحة حسب اللغة

F) app.js
- ضع كل المنطق هنا:
  - قراءة/كتابة اللغة
  - وظائف التوجيه والحماية (auth guard)
  - التحقق من الإيميل الجامعي
  - إدارة المنشورات (قراءة/إضافة/حفظ)
  - إدارة محادثة المعلم (تجريبي)
- اجعل الكود بسيطًا وواضحًا مع تعليقات قليلة بالعربية

3) قواعد عامة:
- لا تستخدم أي مكتبات خارجية.
- اجعل كل صفحة تستورد style.css و app.js.
- تأكد أن الروابط والتوجيهات تعمل عند الضغط على Run.
- اجعل النصوص ثنائية اللغة. استخدم كائن translations داخل app.js.
- بعد الانتهاء: اكتب ملخصًا قصيرًا داخل تعليق أعلى app.js يشرح المفاتيح المستخدمة في localStorage.

ابدأ التنفيذ الآن وأنشئ الملفات كاملة بالمحتوى الجاهز.
