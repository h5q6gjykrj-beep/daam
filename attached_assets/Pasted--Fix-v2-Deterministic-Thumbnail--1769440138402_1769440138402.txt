المطلوب داخل InFeedCampaignCard:

1) استبدل منطق thumbnail الحالي بالكامل بمنطق بسيط وحاسم:
- state:
  const [thumbUrl, setThumbUrl] = useState<string | null>(null);
  const [thumbType, setThumbType] = useState<"image"|"video"|null>(null);
  const [thumbErr, setThumbErr] = useState<string | null>(null);

- useEffect يعتمد على selectedCampaign?.id و selectedCampaign?.updatedAt و attachments/video ids:
  داخل useEffect:
    let cancelled = false;
    setThumbUrl(null); setThumbType(null); setThumbErr(null);

    احسب المرشح:
    - videoId = campaign.video?.id أو أول attachment.kind==="video"
    - imageId = أول attachment.kind==="image"
    - prefer image thumbnail أولاً ثم video (للشكل الأفضل)
    - chosenId + chosenKind

    إذا لا يوجد مرشح: return;

    استرجع blob:
      const blob = await getCampaignAttachmentBlob(chosenId);
      إذا blob == null:
        if (!cancelled) setThumbErr(`blob_null:${chosenKind}`);
        return;

    أنشئ url:
      const url = URL.createObjectURL(blob);
      إذا cancelled:
        URL.revokeObjectURL(url); return;
      setThumbUrl(url);
      setThumbType(chosenKind);

    cleanup:
      cancelled = true;
      if (thumbUrl) URL.revokeObjectURL(thumbUrl);

ملاحظة مهمة: تجنب استخدام useRef list نهائياً.
