⚠️ قيود صارمة:
- مسموح لك تعديل ملف واحد فقط: client/src/pages/feed.tsx
- ممنوع تعديل أي ملف آخر نهائيًا (خصوصًا App.tsx أو Navbar أو أي hooks أو api).

- إذا احتجت ملفًا ثانيًا: توقّف واذكر السبب ولا تعدّل شيئًا.

الهدف: تطبيق تصميم فكرة A المعتمد.

A) إزالة زر "منشور جديد" العريض:
- داخل feed.tsx ابحث عن زر/عنصر نصه "منشور جديد" (أو New Post).
- احذفه بالكامل من الواجهة.
- لا تترك فراغًا كبيرًا مكانه.

B) شريط تحكم Sticky تحت الـNavbar مباشرة (صفّين):
1) صف ترتيب المنشورات (Sort Segmented):
- زرين فقط: "الأحدث" و "الأكثر تفاعلاً"
- أحدهما active افتراضيًا: "الأحدث"
- شكل segmented (كبسولي) منظم، واضح، بدون تشتت.

2) صف التصنيفات (Category Chips):
- Chips أفقية قابلة للتمرير overflow-x-auto
- العناصر: "الكل" (افتراضي)، "سؤال"، "شرح"، "ملخص"، "نقاش"
- عند التفعيل: بنفسجي + نص أبيض
- بدون أيقونات داخل chips

ملاحظة: هذه الأزرار هدفها ترتيب UI فقط. لا تكسر أي منطق فلترة موجود، اربطها بما هو موجود إن كان موجودًا، أو اتركها UI state بدون تعطيل بقية الصفحة.

C) زر عائم ثابت FAB بأيقونة فقط:
- أضف زر دائري fixed لا يتحرك مع التمرير.
- الأيقونة PenLine من lucide-react.
- بدون أي نص.
- الحجم 48-56px.
- اللون #865994.
- RTL/LTR:
  - بالعربية: bottom-right
  - بالإنجليزية: bottom-left
- aria-label:
  - ar: "اكتب منشورًا"
  - en: "Write post"

- Backdrop مع تعتيم + blur خفيف.

- تحتوي:

  - زر إغلاق X

  - زرين: "نشر" و "إضافة صورة" (EN: Post / Add photo)
- دعم RTL/LTR داخل البطاقة والـtextarea.
- الإغلاق بالنقر خارج البطاقة أو زر X.
