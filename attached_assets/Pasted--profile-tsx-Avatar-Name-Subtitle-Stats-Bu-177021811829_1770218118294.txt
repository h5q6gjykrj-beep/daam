المطلوب: إصلاح صفحة profile.tsx لتطابق أسلوب المرجعية: الهوية (Avatar + Name + Subtitle + Stats + Buttons) تكون Overlay فوق الغلاف في المنتصف، مع إلغاء كل أزرار التعديل المتفرقة واستبدالها بأيقونة قلم واحدة تفتح Dialog شامل للتعديل.

⚠️ قيود:
- لا تغيّر أي business logic أو RBAC أو dialogs الحالية (استعمل الموجود قدر الإمكان).
- لا تغيّر routing.
- UI إعادة ترتيب فقط.

====================================================
1) إعادة “الهوية” إلى داخل الغلاف كـ Overlay (مثل المرجعية)
====================================================

في قسم الغلاف (Cover Header):
- اجعل الغلاف يحتوي:
  (A) الصورة + overlay
  (B) Identity Overlay في المنتصف (centered)

Identity Overlay spec:
- container: absolute inset-0 flex flex-col items-center justify-center text-center
- اجعلها تنزل قليلاً للأسفل مثل المرجعية:
  class: "pt-8 md:pt-12" أو "translate-y-6 md:translate-y-8"
- Avatar:
  mobile: w-24 h-24
  md+: w-28 h-28 أو w-32 h-32 (لكن لا تكبر أكثر)
  ring: ring-2 ring-white/25
- Name:
  mobile: text-3xl
  md+: text-4xl (لا تستخدم 5xl/6xl)
  font-bold tracking-tight
- Subtitle (التخصص + الجامعة):
  text-sm md:text-base opacity-85
- Stats line:
  text-xs md:text-sm opacity-75
  الصيغة: "{posts} منشور | {following} يتابع | {followers} متابع"
- Action buttons:
  Row centered (flex gap-3) تحت الـ stats
  - للزائر: [متابعة] [رسالة]
  - للمالك: [رسالة] + (زر/نص) حسب منطقك الحالي، لكن لا تضع “تعديل الملف” كزر كبير هنا.
  استخدم outline soft فقط.

مهم:
- لا تضع الهوية تحت الغلاف. أزل/لا تعرض IdentityCenter خارج الغلاف.

====================================================
2) إلغاء كل أزرار التعديل المتفرقة واستبدالها “بقلم واحد”
====================================================

أزل من UI:
- زر “تعديل الغلاف”
- زر الحذف/سلة بجانبه
- زر “تعديل الملف” الكبير أسفل الهوية

بدلاً منها:
- أضف زر أيقونة واحد (قلم) داخل الغلاف:
  position: absolute top-4 right-4 (RTL مناسب)
  شكل: IconButton دائرية أو مستطيلة صغيرة، شفافة outline
  يظهر فقط للمالك.
  عند الضغط يفتح Dialog واحد شامل: "تحرير الملف".

====================================================
3) Dialog واحد شامل للتعديل (Unified Edit Profile Dialog)
====================================================

عند الضغط على أيقونة القلم:
افتح Dialog شامل يحتوي أقسام/تبويبات داخل dialog (ليس في الصفحة):
A) الغلاف:
  - Upload cover
  - Remove cover
B) الصورة الشخصية:
  - Upload avatar
C) المعلومات الأساسية:
  - الاسم/النبذة/التخصص/الجامعة/المحافظة/الولاية (حسب الموجود عندك)
D) الحساب (إن كان موجود حالياً في الصفحة):
  - تغيير البريد
  - تغيير كلمة المرور
E) حفظ / إلغاء

مهم جداً:
- لا تنشئ منطق جديد من الصفر إن كانت هذه الأشياء موجودة أصلاً في الصفحة كDialogs أو inputs.
- أعد استخدام نفس handlers/sections الحالية، فقط انقلها داخل Dialog واحد.
- إذا توجد Dialogs متعددة حالياً (AvatarPreviewDialog, Edit dialogs):
  اجعل أيقونة القلم تفتح “dialog master” وتضم هذه النماذج داخله.
  (يمكن إبقاء dialogs القديمة في الكود لكن لا تُستخدم من UI).

====================================================
4) Dashboard والكروت
====================================================
- اترك Grid الكروت كما هي (تحسينها لاحقاً).
- المهم الآن: الهوية داخل الغلاف + قلم واحد للتعديل.

====================================================
5) Responsive
====================================================
- على الهاتف: الاسم text-3xl، الأزرار تحت بعض إذا ضاقت (flex-col sm:flex-row)
- لا يوجد قصّ في الهوية داخل الغلاف.

====================================================
التسليم المطلوب:
====================================================
- اعطني مقطع JSX للغلاف بعد التعديل يثبت أن Identity Overlay داخل الغلاف.
- اعطني موضع أيقونة القلم فقط (ولا يوجد أزرار تعديل أخرى).
- أكد أن dialog القلم يحتوي كل أدوات التعديل التي كانت موزعة.
